# Deep-Learning-PAPR-Reduction-Framework-For-5G-NR-Uplink-OFDM
Orthogonal frequency division multiplexing (OFDM) is still the most commonly employed technology in fifth-generation (5 G) new radio (NR) uplinks because of its great spectral efficiency. However, its main drawback is a high Peak-to-Average Power Ratio (PAPR).
